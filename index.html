不是更多 SOP，而是一个会拦截错误的商品增长回路。

这张 Web 原型把三条对话压缩成一个可视化系统：前台增长负责需求、内容和广告；后台商品负责供应、尺码和定价；Loop 方法论负责 Gate、日志、指标和规则更新。

核心对象 ProductCandidate，不是单个 SKU，也不是单条关键词。
核心机制 每一步都有 Gate，没过就不允许进入下一步。
核心目标 让 AI 判断哪些商品值得被人认真管理。

Mutual Challenge

三条线程互相挑战后，系统才完整。

如果只保留任何一条，都会偏：只做广告会忽视供应链，只做供应链会失去需求牵引，只做 loop 会变成抽象框架。

问题关键词机会很好，为什么不能直接投？
挑战如果采购源不稳、尺码只到 US 10、价格没广告空间，PMax 会把供应链问题放大。
系统回应在 CampaignBrief 前加入 Supply Gate、Size Gate、Margin Gate。
问题内容和 Feed 已经生成，为什么还要 ProductFact？
挑战没有事实层，AI 会把供给侧标题、图片感觉和广告卖点混在一起。
系统回应所有 PDP、SEO、Product Feed、Social brief 都只能从 ProductFact 派生。
问题供应稳定就值得维护吗？
挑战如果没有 KeywordCluster 和 Collection Intent，稳定供应也只是库存，不是增长机会。
系统回应ProductCandidate 必须绑定需求、页面角色和渠道角色。
问题尺码窄的款是不是都要淘汰？
挑战它可能适合 product mix、SEO、lookbook 或 petite/slim fit，但不适合 PMax Hero。
系统回应建立 Channel Role，而不是简单 Yes/No。
问题为什么 SOP 还不够？
挑战SOP 只能告诉人怎么做；Gate 可以阻断错误进入下一步。
系统回应每个 Agent 输出都必须有 pass / hold / reject / revise。
问题为什么需要 Log 和 Metric？
挑战没有日志就无法解释决策；没有指标就无法知道规则长期是否有效。
系统回应把每次 Gate 判断回写 ExperimentRun 和 Playbook。

Gate Cockpit

真正让系统稳定的，不是建议，而是拦截。

每个 Gate 都要能输出 pass、hold、reject、revise；并记录为什么过、为什么没过。

Demand Gate

判断是否存在值得做的需求机会。

  • ✓KeywordCluster 有清楚 intent
  • ✓Collection 能承接该需求
  • ✓不是单纯为了上新而上新

Supply Gate

判断供应链能不能承接广告带来的需求。

  • ✓主来源链接有效
  • ✓至少有备用采购路径
  • ✓价格与规格未明显漂移

Size Gate

判断尺码是否能承担对应渠道角色。

  • ✓美码映射明确
  • ✓最大码不足时降级渠道角色
  • ✓前台展示不误导用户

Launch Gate

上线前最后阻断错误进入广告系统。

  • ✓PDP / Feed / 广告 URL 一致
  • ✓价格角色与渠道一致
  • ✓Review Card 已审批

Working Conclusion

这不是最终系统，但它是一个可落地的方向。

建议下一步把这个 HTML 里的结构转成一个真实 MVP：先用 CSV/导入导出跑一个产品，再决定是否做 Web 工作台。

应该先做

  • 一个产品 + 一个 Collection 的完整决策卡。
  • Supply Gate、Size Gate、PDP/Feed Gate、Launch Gate。
  • ProductCandidate、ProductFact、CampaignBrief 三个核心对象。
  • 人工发布，AI 先负责判断、出包和异常提示。

暂时不要做

  • 不要一开始接ERP System、Commerce CMS、Product Feed、Ads 全量 API。
  • 不要让 AI 自动改广告或自动下架。
  • 不要直接覆盖几千个 SKU。
  • 不要把敏感经营数据、供给侧、平台标识 放进外部研究镜像。

一句话版本

Product Growth Lab 需要的是一个 AI Governed Product Growth Loop： 需求机会进入系统前要过 Demand Gate； 商品进入广告前要过 Supply / Size / Margin / PDP / Feed Gate； 上线后通过 ExperimentRun 回写广告表现和供应风险； 表现好的规则进入 Playbook，表现差的规则被降级或淘汰。
